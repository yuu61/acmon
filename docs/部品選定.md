# acmon 物理部品 選定（未確定分のBOM）

Raspberry Pi 4 構成。対象は **README で未確定だった物理部品のみ**。
既決の **Raspberry Pi 4 / ZMPT101B / MCP3208**、およびユーザ所持の **校正済みマルチメータ** は選定対象外。
すべて **ソルダーレス・ブレッドボードで使える**ことを条件に選定し、電気的成立性（分圧比・AC結合コーナー・
VREF電流・バイアス・絶縁境界）を実数で検証済み。検証で出た是正は本表に反映済み（「**訂正反映**」と明記）。

## 1. mains ↔ ブレッドボード境界（最重要）

```
══════════════ 筐体内 / 一次側(100VAC) ════════════════╗   ←─ ここは絶対にブレッドボードに載せない
                                                        ║
  [壁コンセント 100VAC 60Hz]                            ║
        │                                               ║
        ├──→ ★ZMPT101B の AC入力端子 ──(内部トランス=絶縁barrier 1)──┐
        │     ※入力端子は筐体内+ストレインリリーフ必須          │
        │                                               ║         │ 絶縁済み・DCバイアス出力(~1.65V中心)
        └──→ ★降圧・絶縁トランス 一次100V                ║         │
              ・本命: AC-ACアダプタ(プラグ内に封止)      ║         │
              ・代替: HT-1205(裸トランス→要エンクロージャ)║         │
                    │ (絶縁barrier 1)                     ║         │
                    └──→ 二次 ~9VAC(無負荷で~13VAC) ──────╫────┐    │
                                                        ║    │    │
═══════════════════════════════════════════════════════╝    │    │
                                                             ▼    ▼
┌──────────────── ブレッドボード(SELV / 絶縁2次側のみ) ─────────────────┐
│                                                                       │
│  [ZMPT出力]→ MCP3208 ch0 ─┬─ VREF(15) ← 精密3.0V基準(REF3030/LM4040) │
│                           │   + 1µF+0.1µF デカップリング              │
│  MCP3208(DIP-16) ── SPI/3.3V/GND ──→ Pi 4 GPIO (M-Fジャンパ6本)       │
│                                                                       │
│  [トランス二次 ~9-13VAC]→ Rtop(1.8k)→●─→[AC結合4.7µF]→ USBカードLINE-IN│
│                                    Rbottom(100Ω)↓  + 1N4148クランプ   │
│                                              GND(Pi GND共通=OK)        │
│                                                                       │
│  載るもの: MCP3208 / 基準IC / 抵抗(100Ω-2.4k) / セラコン・フィルム /   │
│            1N4148 / DCジャック・3.5mmブレークアウト / USBカード入力線  │
│  最大電圧: ~13VAC rms (SELV, <50VAC) — 100Vはこの板に一切来ない        │
└───────────────────────────────────────────────────────────────────────┘
```

**要点**: mains への絶縁barrierは各経路1段（独立した2トランス）で要件を満たす。両絶縁パスが Pi GND を
共有しても、GND は両トランスの **2次側** にあるので絶縁違反ではない（1枚のブレッドボードで電気的に十分）。
100V 露出部は ①ZMPT 入力端子 ②HT-1205 一次（代替採用時）の 2 箇所のみ＝ **必ず筐体内**。

## 3. 配線・設定メモ（検証で確定した重要数値）

- **VREF（本命REF3030/シリーズ型）**: バイアス抵抗・バッファopアンプ **いずれも不要**。Pi3.3V→3.0Vを
  ドロップアウト~1mVで生成。負荷は最悪1mAで定格25mAの4%＝余裕25倍。
- **VREF（代替LM4040/シャント型）**: バイアス抵抗は **180Ω**（元案270Ωはゼロ余裕の上限。180Ωで無負荷
  ~1.7mA／最悪1mA負荷でも0.7mA以上のシャント余裕）。採用時は **Piの3.3Vレール実測が3.25V以上** を確認。
  精度優先ならシリーズ型REF3030を選ぶ（シャントは避けたいレール由来ノイズを取り込む構成）。
- **VREFデカップリング**: VREFピン直近に **1µF X7R + 0.1µF X7R** の2本並列。
- **分圧比**: 9VAC公称で **Rtop=1.8kΩ / Rbottom=100Ω**（比0.0526→~0.47Vrms）。無負荷13Vrms+15%スウェル
  でも0.97-1.11Vpeakで、クリップ1.78Vpeakに対し37%以上の余裕。**入手後に二次の無負荷Vrmsを実測**し
  `Rtop=100×(Vin/Vtarget−1)` で再選定。
- **AC結合コーナー**: 結合キャップは **4.7µF**（fc=3.4Hz@Zin10k、60Hz基本波減衰0.16%）。1µFは15.9Hzで
  ハイパス減衰3.3%＝THD測定に不適。**DCフリー源なので省略も選択肢**。
- **ZMPTゲイン目標**: 100Vrms入力時 ADC で **~0.9Vrms**（バイアス中心1.65V、上側余裕1.35V＝ピーク上限
  1.35Vに対し0.9Vrmsは1.27Vpeakで収まる）。README記載の1.0Vは1.414Vpeakで **クリップするため不可**。
  バッファ・追加LDOは不要（ゲインポット調整のみ）。**ZMPTはPi3.3Vから給電**し精密基準からは給電しない
  （信号依存のopアンプ電流がVREFを乱す）。スウェルは公称+6%(0.954Vrms)でクリップする構造的制約あり。
- **audio-format**: **必ず `-audio-device hw:1,0 -audio-format S16_LE -audio-rate 48000`**。デバイス名は
  `arecord -l` / `-L` で確認。UCA222は16bit機で、既定S32_LEだとフォーマット不一致→無限1Hz無音再起動
  ループ（`internal/soundcard.go` の captureOnce 再起動ループ）。READMEのS32_LE例をコピーしないこと。
- **絶縁（PASS条件）**: ①ZMPT入力端子を筐体内+ストレインリリーフ ②**HT-1205代替採用時はエンクロージャ
  +一次端子台+ヒューズ+ストレインリリーフがBOM外の必須追加** ③AC-ACアダプタは「AC出力/交流」表記を確認。

### ユーザが既に持っていそうな物（前提＝合計から除外）

- **ブレッドボード本体（EIC-102BJ）** — マルチメータ同様、所持の可能性大。**要確認**。
- **ボード内M-Mジャンパアソート** — 所持の可能性大。**要確認**。
- **AC-ACアダプタ** — 楽器用エフェクタ等で9VAC品を持っていれば流用可。ただし **DC品との誤用に注意**。

### 対象外（再選定しない）

- **既決**: Raspberry Pi 4 / ZMPT101B / MCP3208。
- **所持品**: 校正済みマルチメータ（無負荷二次Vrms・Piレール電圧・分圧後レベルの実測に使用）。

## 5. 発注前チェック（「要確認」一覧）

1. REF3030 の国内単体在庫・価格・最小ロット・送料（マルツ/RS/Mouser 実ページ）。後継 REF30-30 の可能性も。
2. スイッチサイエンス抵抗キット(品番1084)の価格。
3. UCA222 の公称ライン入力Vrms（確定値は最大+2dBVのみ）。
4. AC-ACアダプタの無負荷上振れVrms（入手後マルチメータで実測 → 分圧比を再確定）。
5. ブレッドボード／ジャンパ／ACアダプタの所持有無。

> `-audio-format S32_LE`。本選定では UCA222(16bit) のため **S16_LE 運用が必須**。README既定例とコード
> 既定値を S16_LE に揃えるか注記を足すのが望ましい（別途対応の候補）。
